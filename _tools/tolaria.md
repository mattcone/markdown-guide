Tolaria is an open-source desktop application for managing Markdown-based knowledge bases. It stores notes as plain Markdown files with YAML frontmatter, allowing users to access and edit them with standard tools.

The application includes a block-based editor, wikilinks, and built-in Git integration for version control. Notes are stored locally and can be used without an internet connection.
